Mode d'emploi recommandé

  1. Clique d'abord sur Télécharger une sauvegarde.
  2. Vérifie qu'un fichier nimr-carrosserie-sauvetage-....json est bien téléchargé.
  3. Clique ensuite sur Réparer le cache sans supprimer les données.
  4. Retourne à l'application et recharge la page.
  5. Si les données n'apparaissent pas, utilise l'import sauvegarde de l'application avec le fichier JSON téléchargé.
