# Impressions atelier NIMR SAV

## Documents disponibles

## Fiche de travail technicien

Emplacement : vue **Technicien**, bouton **Imprimer fiche** sur une tâche.

Usage :

- donner une tâche claire au technicien ;
- conserver une trace papier si la tablette n’est pas disponible ;
- noter les pauses, anomalies et signatures.

Contient :

- dossier, client, véhicule, immatriculation, VIN, kilométrage ;
- référence OR ;
- étape atelier ;
- technicien affecté ;
- horaire et durée prévue ;
- consignes, pièces prévues et notes ;
- cases de suivi ;
- signature technicien et validation chef atelier.

## Planning journalier atelier

Emplacement : vue **Planning** ou vue **Technicien**, bouton **Imprimer planning jour**.

Usage :

- briefing chef atelier ;
- affichage papier en atelier ;
- continuité en cas de panne réseau ou tablette.

Les absences et congés ne sont pas mélangés avec les travaux. Ils sont traités comme indisponibilités.

## Fiche de pause / blocage

Emplacement : vue **Technicien**, sur une tâche en pause ou bloquée.

Usage :

- documenter une attente pièces, une attente accord, une panne outil ou un arrêt atelier ;
- faire signer le chef atelier si nécessaire ;
- garder la trace de l’impact planning.

## Ordre de réparation atelier

Emplacement : dossier, bouton **Générer PDF OR**.

Usage :

- dossier atelier principal ;
- synthèse travaux, pièces, durées, validations et affectations.

## Ordres techniciens du dossier

Emplacement : dossier, bouton **Ordres techniciens**.

Usage :

- imprimer toutes les fiches de travail d’un dossier ;
- distribuer les tâches par technicien.

## Contrôle qualité

Emplacement : dossier, section **Qualité & livraison** ou export dossier.

Usage :

- valider les travaux avant livraison ;
- documenter refus, correction ou recontrôle.

## Bon de livraison / PV de restitution

Emplacement : dossier, section **Qualité & livraison** ou export dossier.

Usage :

- restitution client ;
- signature client ;
- réserves et validation contrôle qualité.

## Complément de réparation

Emplacement : dossier, section **Atelier & Compléments**.

Usage :

- anomalie découverte ;
- photos associées ;
- main-d’œuvre et pièces supplémentaires ;
- impact délai et accord requis.

## Procédure papier en cas de panne tablette ou réseau

1. Imprimer le planning journalier depuis le dernier poste disponible.
2. Imprimer les fiches de travail technicien nécessaires.
3. Les techniciens cochent le suivi papier et signent.
4. Le chef atelier conserve les fiches signées.
5. Quand l’application revient, saisir les pauses, notes, blocages et fins de tâches depuis les fiches papier.
6. Lancer une sauvegarde locale ou cloud après rattrapage.
